#####自己不太记得又需要的shortcuts
*   Ctrl+Command+G 选取所有当前词
*   Command + Shift + t 重新打开刚刚关闭的标签页
*   Ctrl + m 括号前后移动光标
*   Shift + Command + Enter 向光标前插入一行
*   Command + Enter 向光标后插入一行
*   Command + K,1 折叠函数一级
*   Command + K,j 取消折叠
*   Command + Opt + [ 折叠
*   Command + X 删除行
*   Command + L 选择行
*   Ctrl + Shift + M 选择括号内容
